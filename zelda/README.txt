//-----------------------MANUAL--------------------------//
1) moving the boat :
	+) A : rotate the boat left
	+) D : rotate the boat right
	+) W : move the boat forward
	+) S : move the boat backward

	+) C : activate Cel shading
	+) O : activate Phong shading
	+) H : show shadow map
	+) N : show normal

	+) ESC: quit application
